## 1. Описание информационной системы

**Название:** Интернет-магазин электроники "TechStore"

**Краткое описание:**  
Учебный пример интернет-магазина, где пользователи могут просматривать товары, оформлять заказы и оплачивать покупки. Администраторы управляют каталогом и заказами.

**Пользователи системы:**
| Пользователь | Что делает |
|--------------|------------|
| Гость | Смотрит товары |
| Покупатель | Оформляет заказы, смотрит свои заказы |
| Менеджер | Обрабатывает заказы |
| Администратор | Управляет сайтом и пользователями |

## 2. Активы данных

| Что храним   | Какие данные                  | Важность         |
| ------------ | ----------------------------- | ---------------- |
| Пользователи | Логин, пароль, email, телефон | Очень важно      |
| Профили      | ФИО, адрес доставки           | Важно            |
| Заказы       | Состав заказа, цены, статусы  | Важно            |
| Платежи      | Данные карт (токены)          | Критически важно |
| Товары       | Название, цена, описание      | Нужно для работы |

## 3. Угрозы

### Конфиденциальность (доступ чужих)

| Угроза          | Как атакуют                      | Опасность    |
| --------------- | -------------------------------- | ------------ |
| SQL-инъекция    | Через поле поиска вводят SQL-код | Очень опасно |
| Перехват данных | Сайт без HTTPS, публичный Wi-Fi  | Опасно       |
| Подбор пароля   | Слабый пароль у админа           | Средне       |

### Целостность (изменение данных)

| Угроза             | Как атакуют                   | Опасность |
| ------------------ | ----------------------------- | --------- |
| Изменение цены     | Перехватывают запрос          | Средне    |
| Подмена заказа     | Угадывают номер чужого заказа | Опасно    |
| Повреждение данных | Ошибка в коде                 | Средне    |

### Доступность (сайт не работает)

| Угроза            | Как атакуют                   | Опасность    |
| ----------------- | ----------------------------- | ------------ |
| DDoS              | Много запросов, сервер падает | Очень опасно |
| Сбой диска        | Закончилось место             | Средне       |
| Ошибка репликации | Сломалась копия базы          | Средне       |

## 4. Примеры атак

### SQL-инъекция

В поле поиска вводят не "ноутбук", а:  
`' OR '1'='1' --`  
Запрос в базе становится:

```sql
SELECT * FROM products WHERE name = '' OR '1'='1' --'
```
